# UVM
> Atomic orchestration enables maximum flexibility

Uni-Virtual Machine(UVM) is an atomic multi-agent orchestration engine. It provides atomic agent ISA and runtime.

- Users easily realize arbitrary orchestration algorithm using U-DSL(which is language for UVM's ISA) and even pure python(through UVM's python API).  
- It supports AI-generated dynamic workflow via reflection mechainsm -- UVM dynamically compiles and executes AI-generated U-DSL.
